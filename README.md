# ImmersiveDemo
Android在4.4版本加入了半透明的界面样式，Android在5.0的时候推出了Material Design的概念。本项目兼容4.4~7.1，以确保状态栏和导航栏在应用中发挥更好的效果。
Demo有两种展示形式
①DrawerLayout+Toolbar
②ActionBar
